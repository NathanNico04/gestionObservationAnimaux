À propos de l'application Projet Framework Web 2 - L3 Informatique - 2025/2026

Cette application permet de gérer des observations d’animaux. Elle offre une interface claire pour consulter, ajouter et organiser les données autour des espèces et de leurs apparitions dans le temps et l’espace.
